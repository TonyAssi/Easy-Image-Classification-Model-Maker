# Easy Image Classification Model Maker
Create a custom image classification model with a few lines of code.

## Installation
```bash
pip install -r requirements.txt
```

## Train Model
Import the model
```python
import modelmaker
```

```python
model = modelmaker.ModelMaker(keywords,
                              num_images,
                              key,
                              dataset_name,
                              model_name,
                              train_epochs)
```
